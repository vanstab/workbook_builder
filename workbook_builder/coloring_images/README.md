# Coloring Images

This folder contains images for the coloring activity pages.

## 📁 Folder Organization

Images are organized into categories for easy management:

```
coloring_images/
├── shapes/      ← Basic geometric shapes (12 included!)
├── animals/     ← Animal outlines (add your own)
├── objects/     ← Common objects (add your own)
├── nature/      ← Flowers, trees, clouds, sun (add your own)
└── custom/      ← Any other images you want
```

**You can add images to ANY of these folders!** The system automatically finds them all.

## 📝 Quick Start (Non-Technical)

**See `_HOW_TO_ADD_IMAGES.txt` for simple, step-by-step instructions!**

## 🎨 Image Format

- **Format:** PNG, JPG, or PDF
- **Recommended:** Line art / outline images suitable for coloring (black outlines on white background)
- **Size:** Any size (will be automatically scaled to fit the page while preserving aspect ratio)
- **Best results:** 400x400 to 800x800 pixels for PNG/JPG

## ⚙️ How It Works

- Each coloring activity page displays 1 image per page (default)
- Images are found in ALL subfolders automatically
- Images are selected in alphabetical order and cycle through as you add more pages
- If no images are found, a friendly message is shown instead
- Supports transparent backgrounds (PNG with alpha channel)

## ➕ Adding Your Own Images

### Easy Way (Drag & Drop):
1. Open the appropriate folder (animals/, objects/, etc.)
2. Drag and drop your coloring images into the folder
3. Done! They'll automatically appear in your worksheets

### Where to Find Images:
- Draw your own and scan/photograph it
- Download from free coloring page websites
- Use Canva or similar tools to create simple line art
- Ask AI tools like ChatGPT/DALL-E for "simple coloring page of [object]"
- Print any image, trace the outlines, scan it back in
- Convert clipart to line art

## 💡 Tips for Best Results

✓ Keep images simple for ages 2-5 (bold, clear outlines)
✓ Avoid very detailed or intricate designs
✓ Use high contrast (black lines on white background)
✓ Test by printing - make sure lines are thick enough to color within
✓ Files starting with `_` or `.` are ignored (hidden files)
✓ Any filename works - images appear alphabetically
